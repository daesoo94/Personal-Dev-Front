PL Laboratory Hompage - Front End
=============
- 앞으로 PL 홈페이지 Front를 만들면서 생기는 과정을 History로 남긴다.

### How To Run? ###
- dependency install
```sh
$ npm install
```
- build and run as develop mode
```sh
$ npm start
```
- build as product mode
```sh
$ npm run build
```
- run built file
```sh
$ npm run dist
```

### Server ###
- AWS EC2 Instance
- PM2
- Node.js
- Express

### Front Dev Environments ###
- Vue.js
- Typescript
- Javascript
- Webpack 4
